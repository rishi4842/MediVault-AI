# 🩺 MediVault AI

AI-powered Medical Image Analysis Platform built for the IDEA2IMPACT Online Hackathon 2026.

## 🚀 Overview

MediVault AI helps users upload medical X-ray images and receive AI-powered analysis in seconds. It provides a clean dashboard, diagnostic summary, medical history, and downloadable PDF reports.

---

## ✨ Features

- 🩻 Upload X-ray images
- 🤖 AI-powered image analysis using Google Gemini AI
- 📊 Diagnostic breakdown
- 🚦 Severity assessment
- 💡 Clinical findings and recommendations
- 📜 Medical history
- 📄 Download PDF medical report
- 🌙 Modern responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Axios
- React Router
- jsPDF

### Backend
- FastAPI
- Python
- Google Gemini AI
- Uvicorn

### Deployment
- Frontend: Vercel
- Backend: Render

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rishi4842/MediVault-AI
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔥 Workflow

1. Upload an X-ray image.
2. Backend sends image to Gemini AI.
3. AI generates structured medical insights.
4. Results are displayed instantly.
5. User can download a PDF report.
6. Report is saved in Medical History.
